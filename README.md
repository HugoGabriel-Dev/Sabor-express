## 🍕 O Sistema Sabor Express

O Sabor Express é um console app projetado para facilitar a gestão de estabelecimentos gastronômicos. Através dele, é possível manter um registro organizado de nomes, categorias e reputação dos restaurantes.

Neste projeto, o foco foi garantir que a lógica de back-end fosse clara, permitindo que novas funcionalidades sejam adicionadas de forma escalável.
